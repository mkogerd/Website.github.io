Today I tried to make a logo! here's the result.

It has nothing to do with my name/initials, but I like that. It's simply an ambiguous symbol - Simple, yet intriguing. Is it a rocket ship? A trophy? The face of a gecko?

Who knows?

I also added it as the site's favicon. It scaled oddly however, I'll have to look into that. Ideally, an icon like this would be drawn using vector graphics so it scales smoothly indefinitely. I tried using an interesting little online application called "Janvas", but it wasn't working for me so I settled with GIMP (hence the asymmetry and failure to neatly "color within the lines").

Edit:

Apparently github does not like displaying my png. Also, I want to give a shout-out to my friend Kirby Mitcham. She hooked me up with a new project: designing a web-page for her photography business. <3

Edit 2:

The logo issue was a simple capitalization issue: I was using "logo.png" to reference "Logo.png". This worked locally for me because Windows filepaths are case-insensitive. However, Github is linux based, which is sensitive about its file-path cases.
